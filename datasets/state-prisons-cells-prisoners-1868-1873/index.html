Prisoners Per Cell in U.S. State Prisons, 1868-1873


Number of cells and average number of prisoners for individual U.S. state prisons (and some large houses of correction and workhouses) in 1868 and 1873. Calculated prisoners per cell.

Get dataset in form/version:
  • Google Sheet published to web (fast, resilient)
  • Microsoft Excel workbook (primary version)
  • Apache OpenOffice Calc spreadsheet
  • Google Sheet (downloadable)
  • Microsoft Excel workbook (secondary, use only if above fail)

Dataset sheets:

  • prisoners per cell: cells, average number of prisoners, and prisoners per cell by state prisons (state, city, prison) in 1868 and 1873

Related datasets:
