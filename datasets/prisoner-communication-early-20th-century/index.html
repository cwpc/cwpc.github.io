Prisoner Communication Policies Early in the 20th Century


Communication policies for prisoners (visiting prisoners and prisoners writing and receiving letters) in U.S. states prisons and in some other countries’ prisons in 1902 and 1912.

Get dataset in form/version:
  • Google Sheet published to web (fast, resilient)
  • Microsoft Excel workbook (primary version)
  • Apache OpenOffice Calc spreadsheet
  • Google Sheet (downloadable)
  • Microsoft Excel workbook (secondary, use only if above fail)

Dataset sheets:

  • communication policies 1902: communications policies in U.S. state prisons (write letters, receive visits); not all state prisons covered. Similar data for prisons in Wurttemberg, Austria, Holland, Great Britain, and Saxony
  • written communication 1912: communications policies in U.S. state prisons (sending letters, receiving letters, receiving magazines and newpapers); covers all state prisons
  • written 1912 summary: summary statistics of communications policies in 1912, with comparison to specific states covered in 1902
